# Dr.fixit
Python script for installing vital software and resolving small difficulties on Linux.
# To use it, do the commands below.
```
git clone https://github.com/ASACHIT/Dr.fixit.git
cd Dr.fixit
pip3 install -r requirement.txt
python3 DRFIXIT.py
```
# Supported Distro
**[1] kali linux**

**[2] Fedora**

**[3] Arch Based Os**
